Welcome

Bitty is a GitHub Gist viewer and editor. It can render Markdown pages and also post new pages to GitHub Gist using the editor provided.

Bitty is unique in a way from other gist viewers out there in that you can customize to an extent the way your page looks. These customizations are stored in a JSON file in the gist that is posted.

Creating a page

To create a page, you can either use the editor here or go to GitHub Gist and create one there.

Using the editor

A page can be created here by clicking New in the top right. If you want to be able to edit the page later, then you should Sign in with GitHub first. Once you have created your page, it will be accessible at URL like http://bitty.io/username/7893883873a705aec69e. The username in the URL can be omitted.

On GitHub Gist

To create the page on GitHub Gist, create a gist with just Markdown content. It is recommended to name the file of your page to bit.md. To customize your page, add another file named bitty.md and follow the specification.

Viewing pages

Just about any Markdown gist will work with Bitty, as long as the Markdown file is the first file.

To view it here:

  1. Take the identifier portion at the end of https://gist.github.com/username/7893883873a705aec69e
  2. Put the identifier at the end of http://bitty.io/. For example: http://bitty.io/7893883873a705aec69e
Note: 7893883873a705aec69e does not point to any valid gist.
